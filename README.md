# Linear Regression Practice in Python

This is a beginner practice project for learning linear regression with Python and scikit-learn.

## What this script does
- loads the FuelConsumptionCo2 dataset
- selects engine size as the input feature
- predicts CO2 emissions
- splits data into training and testing sets
- trains a simple linear regression model
- evaluates results using MAE, MSE, RMSE, and R²
- plots the regression line

## Tools used
- Python
- pandas
- numpy
- matplotlib
- scikit-learn

## Purpose
This project was created as a practice exercise while learning machine learning fundamentals.
